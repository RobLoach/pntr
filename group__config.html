◆ PNTR_ENABLE_MATH

#define PNTR_ENABLE_MATH

◆ PNTR_IMPLEMENTATION

#define PNTR_IMPLEMENTATION

Define PNTR_IMPLEMENTATION in one of your .c files before including pntr.h.

This will let pntr.h know where to implement its functions.

#define PNTR_IMPLEMENTATION
#include "pntr.h"
int main() {
return 0;
}
